# Contributing

Thanks for helping improve this Frappe 16 installation guide.

## How to Contribute

- Keep instructions tested on Ubuntu 24.04 whenever possible.
- Prefer clear, beginner-friendly commands with short explanations.
- Mention whether a step is for development, production, MariaDB, or PostgreSQL.
- Keep PostgreSQL guidance marked as experimental unless the Frappe behavior changes.
- Avoid adding unrelated Frappe or ERPNext customization topics to this repository.

## Pull Request Checklist

- Commands are copy-paste friendly.
- Markdown headings and code fences render correctly.
- Troubleshooting notes explain the cause and the fix.
- New scripts are safe to run and avoid destructive actions.
- README keywords stay natural and useful, not stuffed.
